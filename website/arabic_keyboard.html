Anda juga dapat mengetik menggunakan transliterasi Latin. Contoh: "a" → ا, "t'" → ث, "H" → ح, "S" → ص, "D" → ض, "T" → ط, "Z" → ظ.

Petunjuk Penggunaan Keyboard Arab
Mengetik langsung dengan keyboard komputer:
  • Untuk huruf tebal (emphatic), ketik huruf kapital: H, S, D, T, Z
  • Untuk menambah tanda diakritik, ketik apostrof: t', H', d', s', S', T', g'
